Основы HTML

Что такое HTML?

HTML (HyperText Markup Language)-это язык разметки для создания веб-страниц и веб-приложений.

Основные теги HTML
  • <"!DOCTYPE html"> - указывает тип документа и версию HTML
  • <"html"> - определяет корневой элемент документа
  • <"head"> - содержит метаданные документа
  • <"title"> - устанавливает заголовок документа
  • <"body"> - содержит содержимое документа
